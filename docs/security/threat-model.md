# Threat Model

Tijoir focuses on practical protection for vendor credential exchange workflows.

## In Scope

- Replacing email, chat, tickets, documents, and screenshots for secret exchange.
- Preventing reused one-time share links.
- Preventing cross-organization secret access.
- Preserving append-only audit evidence.
- Revoking vendor access during offboarding.

## Out of Scope for MVP

- Full zero-knowledge encryption.
- Mandatory MFA.
- Enterprise SSO and SCIM.
- Compromised end-user devices.
- Malicious cloud provider or compromised cloud project owner.
